# AirBnB-Data-Analysis
A begginer level Data Analysis Project - On Air BnB Dataset taken from Kaggle and EDA was performed on the dataset.
Performed Data Manipulation on the dataset to make it more consistent and accurate for analysis,
Changed the data type of the columns that were required for the visualization
Drew insights by solving four different queries:- 1)What is the distribution of listing prices? 2)How are different types of rooms distributed? 3)How are listings distributed across different neighbourhoods? 4)What is the relationship between price and room type?
